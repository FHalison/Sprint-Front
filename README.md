# SoulUp Mobility

#Descrição do Projeto

O SoulUp Mobility é uma solução digital desenvolvida para permitir a conversão de pontos acumulados em créditos para transporte público, incentivando a mobilidade urbana sustentável e reduzindo impactos ambientais.

A plataforma foi criada com foco em acessibilidade, praticidade e sustentabilidade, oferecendo uma interface moderna, intuitiva e responsiva para diferentes dispositivos.

---

#Objetivo

O objetivo do projeto é conectar benefícios digitais ao transporte público, permitindo que usuários utilizem pontos acumulados para financiar total ou parcialmente o uso de mobilidade urbana.

---

#Funcionalidades

- Navegação responsiva
- Conversão de pontos em créditos
- Dashboard do usuário
- FAQ interativo
- Formulário de contato
- Simulação de impacto ambiental
- Interface moderna e intuitiva
